CSS Color Cheatsheet

This page will give a brief set of several color options for a CSS file

  • color:
  • background-color:
  • Keyword Color
  • RGB Color
  • Hexadecimal Color
  • HSL Color
  • Opacity

Color

Syntax: color: black;

The color element directly affect the foreground color. In this example the text is displayed as black

Background Color

Syntax: background-color: AntiqueWhite;

The color element directly affect the background color. The above example sets the background color to AntiqueWhite

Keyword Colors

Syntax: color: black;
Syntax: background-color: blue;

Keyword colors are just as described. The color is a specific keyword that can be typed into CSS. It is very limited though so use if for more general colors.

RGB Color

Syntax: color: rbg(0, 255, 123);

The RBG color element is "rbg(x, x, x)" where x is a numeric value from 0 to 255. The first number is the amount of red, the second is the amount of green, and the third is the amount of blue.

Hexadecimal Colors

Syntax: color: #FF00A0;

The hexadecimal color element is "#xxxxxx" were x is hexadecimal number from 0 to F. The format is either a 3 or 6 digit combination of hexidecimals characters. The first 2/1 characters are the amount of Red, the second 2/1 characters are the amount of green, and the last 2/1 characters are the amount of blue. 00/0 is white and FF/F is full color.

HSL Color

Syntax: hsl(249, 70%, 65%);

The hsl color is "hsl(x, y, y)" element stands for hue, saturation & lightness. The hue or, x in the example, is a numberic number from 0 to 360 representing the angle on the color wheel (0/360 = Red, 120 = Green, 240 = Blue). The saturation, or the first y, is a % from 0 to 100%. The closer to 100 the more rich the color is and the closer to 0 the duller the color is with 50% being normal. The last digit is the lightness, or the second y, is a % of how light the color is. The closer to 100 the value is the whiter the color will be and the closer to 0 the closer to black the color will be. 50% is normal

Opacity

Syntax: color: transparent;
Syntax: color: rgba(0,0,0,.5);

Syntax: color: #0007; / color: #00000077;

Syntax: color: hsla(0, 100%, 0%, 0.5);

Opacity affects how transparent an element is. There are several ways that are outlined above. The added letter to the rgb and hsl stands for alpha which represents opacity.

  • Keyword: The only option for opacity is transparent, making the element completely clear.
  • RGB: The syntax is changed to rgba and a fourth digit is added which is a digit between 0.0 to 1. with 0 being transparent and 1 being completely opaque.
  • Hexadecimal: Depending if the there are 6 digits or 3 digits, an additional 2 or 1 more set of hexadecimal characters are added to the end with 77 / 7 being half transparent.
  • HSL: Similar to RGB the syntax is changed to include an "a" to the end and a fourth digit is added with the same parameters as rgb.
